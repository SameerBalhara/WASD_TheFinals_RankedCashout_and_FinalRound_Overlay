# WASD_TheFinals_RankedCashout_and_FinalRound_Overlay
Applying scoreboard math to solve for who is alive and dead in The Finals Ranked Cashout and Final Round gamemodes (by default, this information is otherwise hidden)

**Instructions**
1. Press the green "Code" button in the top right and select "Download ZIP".
2. Unzip the file and only keep the following files: "WASD_RankedCashout_and_FinalRound_Overlay.exe", "settings.json", "README.md"
3. Customize settings.json with any preferred keybinds and save this file prior to launching the exe. Ensure settings.json and the exe always exist in the same folder/directory
